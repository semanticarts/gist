### Major Updates

- Set ranges of `gist:containedText`, `gist:uniqueText`, `gist:description`, `gist:name` and `gist:symbol` to `rdf:PlainLiteral`, which encompasses string literals with or without language tags. Changed corresponding restrictions on `gist:Text`,`gist:ID`, `gist:Tag` and `gist:MediaType` from `xsd:string` to `rdf:PlainLiteral`. Issue [#1302](https://github.com/semanticarts/gist/issues/1302).
